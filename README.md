# Projeto de Sistema Completo

    Neste projeto estou desenvolvendo inicialmente o back end, no qual irá fazer um crud de algumas informações. Assim que acabar o back end, irei fazer o font com React.

# Tecnologias e ferramentas utilizadas:
    
    - Node JS
    - Express
    - TypeScript
    - Typeorm
    - Postgres
    - Docker

    Utilizando também alguns conceitos de Código Limpo e Clean Arquitecture.

# Requisitos Funcionais

        1 - Adicionar novos usuários
        
        2 - Listar usuários existentes

        3 - Editar usuários

        4 - Deletar usuários

# Como rodar a API

    - Primeiramente você deve ter o Docker e o Docker Compose instalado em sua máquina.

        * Subindo todos os containeres:

            docker-compose up -d

        * Matando os containeres:

            docker-compose down
